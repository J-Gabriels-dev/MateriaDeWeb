1. O que é CSS

CSS (Cascading Style Sheets) controla a aparência visual dos elementos HTML. Uma regra CSS tem duas partes: o seletor (qual elemento) e a declaração (propriedade + valor).

4. Cascata e Especificidade

Quando várias regras atingem o mesmo elemento, vence a de maior especificidade. Com especificidade igual, vence a declarada por último.

Cálculo A · B · C

  • A — ID selectors (#id)
  • B — Class, atributo, pseudo-class (.classe, :hover)
  • C — Type, pseudo-element (h1, ::before)
→ 0 0 0 (menor) h1 → 0 0 1 .classe → 0 1 0 #id → 1 0 0 (maior)

Pseudo-classes comuns

:link · :visited · :hover · :active · :focus · :first-child · :nth-child(n)

Pseudo-elementos comuns

::before · ::after · ::first-letter · ::first-line · ::selection

5. Herança

Filhos herdam certas propriedades do pai. color e font-family são herdados. border e padding não são.

10. Sistemas de Layout

Block vs Inline

Block: ocupa a linha inteira, empilhados verticalmente (div, p, h1).
Inline: ocupa só o espaço necessário, fluem horizontalmente (span, a).
A propriedade display muda o comportamento: block | inline | flex | grid.

Posicionamento (position)

ValorComportamento
staticFluxo normal (padrão)
relativeRelativo à sua posição normal
absoluteFora do fluxo; posicionado em relação ao pai
fixedFixo na janela do browser
stickyNormal até certo ponto de scroll, depois fixo

Float Layout

float: left ou float: right move o elemento para a borda do bloco. O pai precisa de overflow: hidden para conter os filhos flutuantes. Considerado legado — prefira Flexbox ou Grid.

Flexbox (display: flex) — 1 dimensão

Organiza itens em linha ou coluna. O container ajusta o tamanho dos itens para preencher o espaço.
